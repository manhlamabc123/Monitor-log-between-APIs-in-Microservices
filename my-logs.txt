1. Go to https://www.django-rest-framework.org/tutorial/quickstart/

2. Create python environment: python3 -m venv microservices

3. Activate environment: source microservices/bin/activate

4. Install Django & Django Rest Framework
- pip install django
- pip install djangorestframework

5. Start django project:
- django-admin startproject admin

6. Create Dockerfile

7. Create requirements.txt
